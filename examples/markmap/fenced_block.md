# Markmap in Markdown

Here is a markmap rendered as a fenced code block:

```markmap
# Operating Systems
## Linux
- Ubuntu
- Arch
- Fedora
## macOS
- Homebrew
- Xcode
## Windows
- WSL
- PowerShell
```

The markmap above shows OS options as a tree.

You can also use it for quick outlines:

```markmap
# Rust
## Ownership
- Borrowing
- Lifetimes
## Concurrency
- Threads
- Async/Await
## Ecosystem
- Cargo
- crates.io
```
